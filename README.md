# Autonomous Warehouse Automation using Siemens PLC and Factory I/O

## Overview

This repository contains an industrial warehouse automation project developed as part of the MSc Automation and Robotics programme at Anglia Ruskin University.

The project demonstrates the design and implementation of an automated warehouse capable of transporting, sorting, storing, and retrieving cargo using Siemens PLCs programmed in TIA Portal and simulated in Factory I/O.

---

## Features

* Automated cargo transportation
* Height-based object sorting
* Pick-and-place automation
* Automated palletizing
* 54-location warehouse storage system
* Automated loading and unloading
* Sequential process control
* Sensor and actuator integration
* Real-time industrial simulation

---

## System Workflow

1. Cargo enters the warehouse.
2. Sensors detect object dimensions.
3. Cargo is classified according to height.
4. Pick-and-place mechanisms transfer the cargo.
5. Pallets are automatically generated.
6. Storage locations are assigned within the warehouse.
7. Cargo is retrieved and dispatched when required.
